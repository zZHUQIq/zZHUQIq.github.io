s# 大脑的基本结构
1. **解剖平面**：矢状面、冠状面、横断面【人】；矢状面、水平面、冠状面【鼠】![[Pasted image 20240229095902.png]]![[Pasted image 20240229095920.png]]
2. **大脑分区**
	1. 结构：包含五个叶：额叶、顶叶、枕叶、颞叶、岛叶【藏于额叶和颞叶里】
	2. 功能分区
		1. 运动皮层(中央沟的前面)：脑机接口
		2. 感觉皮层(中央沟的后面)
3. **神经细胞的电位** 
	1. 动作电位：宽度1-2ms，传递时间2-4ms，因此当两个动作电位时长超过8ms时，存在多突触的可能性。![[Pasted image 20240414164026.png]]
	2. 突触可塑性：
		1. PPR(Paired Pulse Ratio——测突触前细胞的可塑性)——验证突触前递质释放概率变化的电生理指标——【PPR越大，突触前神经元释放递质的概率低】；
		2. STDP(放电时间依赖的突触可塑性)：高频刺激/药物(内啡肽)后测突触后的可塑性(LTP+LTD)。![[Pasted image 20240414164227.png]]
4. **神经细胞电突触的测试**：生物素的双向扩散+A、B细胞的互相兴奋+兴奋传递更快无突触延迟

# 多通道电极记录技术
1. 细胞外记录——Cell attach(金属丝靠近细胞)，只可测单个细胞的动作电位和集中多个细胞的场电位
	1. MEA(微电极阵列)：表面信号的探测——如视网膜刺激的探测
	2. Microwire Array(微丝阵列)：电极间距400um对细胞密度不高的，100-200对密度高的——一开始插电极的深度可以稍微不足，后续可调整突破免疫反应产生的结痂到达指导深度；
	3. Utah Array(犹他阵列)：只有针尖可导电；
	4. 柔性基底电极
	5. 硅基电极阵列
2. **多电极技术**【探索与行为相关的神经元】
	1. **四电极分离**：神经元到四个电极的强度比例——辨别神经元放电时的身份，通过波形、波宽和拖尾也可区别神经元，比如Gaba、锥体神经元较宽，较窄且频率高的为中间神经元【滤波器要滤到300hz以下的，2ms的波宽代表500hz】![[Pasted image 20240415135835.png]]
	2. 单/双神经元放电间隔分析(ISI)：放电是否规律(单)、分析两个神经元的相互关系(上下级、同步放电)——互相关分析![[Pasted image 20240415140706.png]]
	3. **局部场电位分析**：根据波形分(如α、γ等)一群细胞的状态，监控睡眠状态(如睡着有ripple涟漪)
# 组装体(同学)
1. 流程：获得聚集小球体——诱导因子诱导成不同皮层细胞（免疫组化验证）——两类小球放在一块组装(细胞的自组装)
2. 形态学检测：组装后进行荧光标记或者逆行病毒标记失踪(两个细胞标记不同的荧光)看组装是否成功（共定位）以及投射情况（免疫组化来看投射细胞的类型）
3. 功能检测：光遗传相关蛋白在上游激活后，检测下游的钙活动+膜片钳(光激活上游，检测下游电信号)
4. 组装体——神经发育障碍组装体的效果不太好，钙信号活动弱、肌肉收缩的调控+组装体移植到小鼠中进行药物检验
